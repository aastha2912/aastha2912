## Research & Publications

**Agentic LLM Pipeline for Automated Vulnerability Detection and Repair** *(forthcoming, June 2025)*  
End-to-end agentic pipeline for vulnerability detection and patch synthesis across multi-turn code generation contexts. Agent orchestration, context capture instrumentation, eval against real bug corpora.

---

**ARVO Patching Agent Evals · AIxCC AFC Archive** · [→ repo](https://github.com/aastha2912/aixcc-afc-archive/tree/main/crs/arvo-patching-agent)  
Integrated the Theori patching agent with the ARVO dataset and ran structured evals: detection accuracy, patch quality, and the core question — how much is reasoning vs. memorization.

---

**CVE Vulnerability Reproduction Agent** · [→ repo](https://github.com/aastha2912/cve-vuln-reproduction)  
Agent that takes a CVE description and generates a runnable Docker reproduction environment — Dockerfile, startup scripts, validation commands. GPT-4 drives environment generation; a second pass confirms the exploit is live inside the container.

---

**MT-Sec: Benchmarking Correctness and Security in Multi-Turn Code Generation** · [→ arXiv](https://arxiv.org/abs/2510.13859) *(2025)*  
Built the benchmark, eval pipeline, and agentic detection layer across 2,376 tasks and 27 CWEs. Key finding: even state-of-the-art models show a 20–27% drop in correct & secure outputs from single-turn to multi-turn — and agents make it worse, not better.
